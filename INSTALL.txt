Installations instructions
==========================

Go to the settings page of this plugin to get an example of the command line you need to configure.

The options you can configure in the command line are:

1. secret (required): this is to prevent unauthorized execution of the command line, a unique code will be generated by the system for your installation.
2. interval (required): the interval of the cron, also see http://docs.elgg.org/wiki/Cron for more information.
3. host: the HTTP_HOST of your Elgg installation.
4. https: set to 'On' if your site uses HTTPS.
5. memory_limit: with this setting you can configure the amount of memory PHP is allowed to use
